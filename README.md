## Installation
To set up the PING WINN GANG system, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/roman181/20.01.git
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
Once installed, you can run the system using the command:
```bash
npm start
```
Ensure to configure the environment variables as outlined in the `.env.example` file.
